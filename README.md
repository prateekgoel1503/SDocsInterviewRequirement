# SDocsInterviewRequirement

Salesforce project for SDocs interview requirements.

## Project Structure

This project follows the Salesforce DX project structure with source format.

## Getting Started

1. Authenticate with your Salesforce org:
   ```bash
   sf org login web
   ```

2. Deploy the metadata to your org:
   ```bash
   sf project deploy start
   ```

## Development

- Source code is located in the `force-app/main/default` directory
- Use Salesforce CLI for deployment and development operations
